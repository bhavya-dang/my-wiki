# My Dev Setup 2025

So, I recently got a new laptop and revamped my development setup. I thought it would be a good idea to share my setup and the tools I use on a daily basis.

![alt text](../../../public/desk-setup-2025.png)

The desk/cabinet is a bit old and you can see some wear and tear but it gets the job done till I upgrade. I have a pretty minimal setup with just the essentials. I like to keep my workspace clean and clutter-free.

Here's a quick overview of my setup:

- **Hardware**:
  - **Laptop:** [MacBook Pro M4 Pro 16-inch](https://www.apple.com/in/shop/buy-mac/macbook-pro/16-inch-space-black-standard-display-apple-m4-pro-with-14-core-cpu-and-20-core-gpu-24gb-memory-512gb)
  - **Monitor:** HP E233 23-inch (got this from my office, not the best but works fine for now)
  - **Keyboard:** [Royal Kludge RK68](https://meckeys.com/shop/keyboard/65-keyboard/rk68-rgb-gateron-mechanical-switches/)
  - **Mouse:** [Logitech G402](https://www.logitechg.com/en-nz/products/gaming-mice/g402-hyperion-fury-fps-gaming-mouse.html)
  - **Headphones:** [Sony WH-1000XM4](https://www.sony.co.in/electronics/headband-headphones/wh-1000xm4)
  - **Microphone:** [Fifine AM8T](https://www.amazon.in/FIFINE-Microphone-Podcasting-Voice-Over-AmpliGame-AM8T/dp/B0C74GYW3J)
  - **Docking Station:** Honeywell USB-C Docking Station (got this from my office as well)
  - **Laptop Stand:** [Zebronics Laptop Stand](https://shop.zebronics.com/products/zeb-ns2000) (not the best but does the job)
- **Software**:
  - **OS:** macOS Sequoi (15.6.1)
  - **Code Editor:** [Visual Studio Code](https://code.visualstudio.com/)
  - **AI Copilot:** [Ollama](https://ollama.com), [Continue](https://continue.dev)
  - **Terminal:** [Warp](https://www.warp.dev/)
  - **Shell:** Zsh with [Oh My Zsh](https://ohmyz.sh/)
  - **Package Manager:** [Homebrew](https://brew.sh/)
  - **Browser:** [Zen](https://zen-browser.app/), [Chrome](https://www.google.com/chrome/), [Brave](https://brave.com/)
  - **Note-taking and Management:** [This wiki](https://wiki.bhavyadang.in/), [Notion](https://www.notion.so/) (but I really wanna try [Obsidian](https://obsidian.md/))
  - **Version Control:** [Git](https://git-scm.com/)
  - **Spotlight Search:** [Raycast](https://www.raycast.com/)
  - **Design:** [Figma](https://www.figma.com/)
  - **Music:** [Spotify](https://www.spotify.com/)
  - **Task Management:** [Todoist](https://todoist.com/)

I had the following running on my pi:

- [Docker Compose](https://docker.com): Container Orchestration
- [Jellyfin](https://jellyfin.org/): Media Server
- [Pi-hole](https://pi-hole.net/): Network-wide Ad Blocker
- [Portainer](https://portAINER.io/): Docker Container Management
- [Uptime Kuma](https://uptimekuma.com/): Self-hosted Monitoring Tool
- [CasaOS](https://casaos.org/): Home Server Management
- The ARRs:
  - [Radarr](https://radarr.video/): Movies
  - [Sonarr](https://sonarr.tv/): TV Shows
  - [Prowlarr](https://prowlarr.com/): Indexers
- [FlareSolverr](https://github.com/FlareSolverr/FlareSolverr): Bypass Cloudflare
- [ByteStash](https://github.com/jordan-dalby/ByteStash): Snippet Manager
- [Tailscale](https://tailscale.com/): VPN

Going forward, I will be storing all my dotfiles in a Github repo so that I can easily set up my environment on any new machine.
